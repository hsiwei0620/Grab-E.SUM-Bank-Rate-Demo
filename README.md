### Environment Setting:
* apache Tomcat v9.0    
* jre1.8.0_191
### Import external file:
* jquery-3.3.1.js
* jscal2.js (datepicker)
* shieldui line chart [sample demo](https://demos.shieldui.com/web/line-chart/basic-usage)
